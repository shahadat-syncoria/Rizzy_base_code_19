How to Configure Purolator Shipping Instance in Odoo

Go to your Odoo database dashboard. Clink “Settings”

From settings select “Purolator Delivery Methods” under “Shipping”

Select Purolator, CN

Enter Credentials to configure Purolator Shipping Instance

Select the delivery method in Quotation/ Sales Order and select “GET RATE”

The rates will appear in the Quotations/Sales Order

Validate the Quotation/ Sales Order by clicking “VALIDATE”

Now print the shipping label by selecting “PRINT SHIPPING LABEL” and clink “OK”

Select “Purolator, CN” and press “Ship” button in Odoo website to complete the order
